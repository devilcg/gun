# skpine Chrome Extension 설치 안내

## 로컬에서 로드(개발자 모드)
1. Chrome에서 `chrome://extensions`로 이동합니다.
2. 오른쪽 상단의 **개발자 모드**를 켭니다.
3. **압축해제된 확장 프로그램 로드**를 클릭합니다.
4. 이 레포지토리의 `extension/` 폴더를 선택합니다.

## 사용 방법
- **집중 모드 토글**: 네이버 쇼핑 상세 페이지에서 집중 모드를 켜고 끕니다.
- **최저가 분석**: 상품명/가격/이미지를 수집해 `skupine.ai` 분석 페이지를 엽니다.

## 참고
- 확장 프로그램 업데이트 시 `chrome://extensions`에서 새로고침 버튼을 눌러 적용합니다.
